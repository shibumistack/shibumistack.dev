> Refined simplicity for shipping web apps.

Shibumi Stack is a small, opinionated path for building durable apps with Bun, Hono, Drizzle, Alpine, and Zod.

Small pieces chosen with care. Clear seams. No hidden runtime. Built to age well.

```sh
bun create shibumi@latest my-app
cd my-app
bun dev
```
